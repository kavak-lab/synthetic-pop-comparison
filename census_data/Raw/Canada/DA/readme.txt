Counts for each socio-demographic variable at the dissemination area geography.
Datasets are organized by variable. Actual column names for each variable are stored in the corresponding header files.
The geographic boundary files are included. This is a shapefile of the geographic boundaries of all dissemination areas in Greater Vancouver Area. The tabular data can be joined to the geographic boundary files.
